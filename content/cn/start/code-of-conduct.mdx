# 我们的承诺

作为成员、贡献者和组织者，我们承诺为所有人提供无骚扰的社区参与体验，不论年龄、体型、有形或无形的残疾、族群、性特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、种姓、肤色、宗教或性认同和取向。

我们承诺以促进开放、欢迎、多元、包容和健康社区的方式行动和互动。

## 我们的标准

有助于为社区创造积极环境的行为示例包括：

<Process>

对他人表现出同理心和善意 

尊重不同的意见、观点和经历

给予也优雅地接受建设性反馈

承担责任，向受我们错误影响的人道歉，并从中学习

不仅关注我们作为个人，还关注社区整体的最佳利益

</Process>

不可接受的行为如：

<Process>

使用性暗示的语言或图像，以及任何形式的性注意或挑逗

恶意挑衅、侮辱性或贬低性评论，以及个人或政治攻击

公开或私下骚扰

未经允许泄露他人私人信息，如地址或邮箱

其它在专业场合可能被视为不适当的行为

</Process>

## 未成年人

如果你未满 18 岁，必须获得父母或监护人的同意才能参加我们的活动，因此，你的父母或监护人需要同意这些条件。在参加每次 Kernel 活动之前，须提交由父母或监护人签署的同意书。你承认，即使作为未成年人，也必须阅读并理解这份行为准则，并有能力做到这一点，想要遵守这里描述的标准和要求。

# 谁对此负责？

组织者（stewards，以下根据语境，有时译为“组织者”，有时译为“管理者”）负责澄清和执行我们的可接受行为标准，并将对他们认为不当的、具有威胁性的、冒犯的或有害的行为采取适当且公正的纠正措施。

组织者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交（commits）、代码、wiki 编辑、议题（issues）和其他贡献，并在适当时沟通调整的原因。

## 适用范围

此行为准则适用于社区的所有空间，当个人正式代表社区在公共空间活动时也同样适用。这类代表行为如使用官方邮箱、通过官方社交媒体账户发布信息，或作为指定代表参加线上线下活动等。

可通过 moderation@kernel.community 向管理者报告欺凌、骚扰或其他不可接受的行为。

所有投诉将被迅速和公正地审读与调查。

所有组织者都有义务尊重任何事件报告者的隐私和安全。

## 社区影响指南

在确定任何被认为违反行为准则的行为的后果时，管理者将遵循以下社区影响指南：

<Process>

<Box>

**纠正**

社区影响：在社区中使用不当语言或其他被认为不专业或不受欢迎的行为。

后果：社区管理者将提供私下书面警告，明确违规性质及解释该行为为何不当。可能会要求公开道歉。

</Box>

<Box>

**警告**

社区影响：单次违规事件或者一系列违规行为。

后果：警告并说明持续行为的后果。在指定时间内，不得与涉事人员（包括未经准许与行为准则执行人员）进行互动。这包括避免在社区空间以及像社交媒体这样的外部渠道进行互动。违反这些条款可能会导致临时或永久禁令。

</Box>

<Box>

**临时禁令**

社区影响：严重违反社区标准，包括持续不当行为。


后果：在指定时间内，临时禁止与社区进行任何形式的互动或公共沟通。在此期间，不得与涉事人员进行公开或私下互动，包括未经请求与行为准则执行人员的互动。违反这些条款可能会导致永久禁令。

</Box>

<Box>

**永久禁令**

社区影响：展示违反社区标准的行为模式，包括持续不当行为、骚扰个人或对个人或群体的侵犯或贬低。

后果：永久禁止在社区内进行任何形式的公共互动。

</Box>

</Process>

# 核（Kernel）的关怀

<InfoBlock>
 
和谐照顾彼此，

我们共同营造 

信赖的空间， 

开放——充满欢乐！——为了游戏。

</InfoBlock>

Kernel 建立在我们相互关怀的责任之上。关怀是我们所做一切的首要：我们首先为他人付出劳动；带着关注、礼貌和尊重去倾听；言辞温和；并以展现爱与照顾的方式分享或回应。

作为 Kernel 成员或校友，**你有关怀的义务（Obligation）**。

在许多现代宪法中，都有这样的义务：例如，南非要求其公民关怀老年人和被监禁者。这类义务与权利的执行大不相同——难以在静态文本中描述如何履行它们。一个可能的答案是“启发”。当我们的领导者以身作则，按照履行我们集体义务的方式行动和生活时，普通公民也会被鼓励做同样的事。


我们每个人都必须努力意识到我们在这里所同意的字句，并了然在其后激发这些字句的精神。我们看重我们的个人责任，因为这是一种好的生活方式。我们这样做是为了培养与他人持续、健康的关系，因为只有当我们一起好好生活时，我们才能完整地生活。
